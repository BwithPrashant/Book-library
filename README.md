# Book-library
This is a golang project which exposes Rest API's for book management in a library

# Rest Endpoints

- Add book
  endpoint : /books
  method : POST

- Modify book
  endpoint : /books/{id}
  method : PUT


- Delete book
  endpoint : /books/{id}
  method : DELETE
  
 
- Get book by ID
  endpoint : /books/{id}
  method : PUT
  
  
- GetAll book
  endpoint : /books
  method : PUT

- Get books by filter
  endpoint : /books?filter1=param1,param2&filter2=param3
  method: GET
